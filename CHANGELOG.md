# Changelog

## 2026-07-22

- Published workflow V7 as the active/default standalone 16-file payload; V6
  remains predecessor-only evidence for bounded migration.
- Replaced remote-oriented ordinary completion with lean local-first project
  continuity, proportional validation, and scoped verified local Git saving.
- Added a skill-owned global activation controller. Manual remains the default;
  users may explicitly opt into Automatic activation or return to Manual, and
  installing or updating the skill preserves that choice.
- Added transactional exact 15-to-16 V7 installed-payload upgrades while
  continuing to refuse missing non-additive managed paths.
- Validated the release matrix on Windows PowerShell Core 7.6.3 and Linux
  PowerShell Core 7.6.2. macOS remains unavailable and unvalidated.
- Removed tracked predecessor backup configuration from the current public
  tree. Migrated-project legacy GitHub helpers remain inert and unowned by V7.
